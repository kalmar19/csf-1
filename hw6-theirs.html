Problem 3: For Loops

Python provides the for loop that can be used to simplify programs that contain iteration. It is a simpler way to use iteration than using while loops.
The general form of a for statement is:
for variable in sequence:
code block

Problem 4: Creating Lists

A list is an ordered sequence of values, where each value is identified by an index. A list can contain values of any type. An empty list is written as [].
For example, the code,
myList = [1, 2, 3]
print myList
produces the output,
[1, 2, 3]
Create a list containing the values 1, 2.2, 3, 4.2 and then write Python code using a loop, that prints out each element in the list and prints out the average value of all of the elements within the list.

list3 = list1 + list2

list1.append(list2)
print list3
print list1
produces the output,
['a', 'b', 'c', 'd', 'e']

Problem 7: Creating a Dictionary

A dictionary is like a list, but the indices can be almost any type unlike a list. A dictionary is a collection of key-value pairs. A key and its value are separated by a colon. And the key value pairs are separated by commas. The key and value pairs are listed between curly brackets "{}".

Create a dictionary containing all the months in the year. Set the keys according to the months number in the year.
For example, months[1-12] are the keys and January-December are the values.
